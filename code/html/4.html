Hyperlink

Untuk membuat hyperlink atau yang sering disebut link kita hanya perlu tag a saja. Tag a adalah singkatan dari anchor

Agar lebih jelas cara menuliskan tag a ini mari kita lihat kodenya.



<a href=""></a>


Jika kalian menemukan sebuah tag dengan spasi lalu ada sebuah tag lagi, itu adalah Attribute. Kita katakan saja Attribute tadi itu atribut saja. Untuk memperjelasnya mari kita lihat kodenya lagi.



href=""


Nah href tadi juga disebut atribut.

Fungsi atribut href adalah untuk menyalakan suatu link. Di dalam href kita bisa menuliskan External link dan Internal link.External link adalah jenis href yang mengarahkan kita ke website. Sementara Internal link adalah jenis href yang mengarahkan kita dari satu file ke file lain.

Kita masuk ke contoh External linknya dulu.



<!-- Contoh External Link --> <a href="https://www.youtube.com/">Masuk ke youtube.</a>


Hasil :

Masuk ke youtube.

Lanjut masuk ke contoh Internal linknya.



<!-- Contoh Internal Link --> <a href="3.html">Kembali ke materi sebelumnya</a>


Hasil :

Kembali ke materi sebelumnya



Tag a juga bisa diberi atribut yaitu, target. Target dibedakan menjadi 4 yaitu :

  • _blank
  • _self
  • _top
  • _parent

Untuk _top dan _parent tidak akan kita bahas karena _top dan _parent berhubungan dengan frame, sementara kita belum belajar mengenai frame.

Kita bahas _blank dan _self dulu. Default nya jika tidak diberi atribut target adalah _self. Langsung saja kita bahas. _self untuk membuka link di tab yang sedang kalian gunakan. Sementara _blank untuk membuka link di tab baru.

Contoh _blank dan _self



<!-- Contoh _blank --> <a href="https://www.youtube.com/" target="_blank">Masuk ke youtube.</a> <!-- Contoh _self --> <a href="https://www.youtube.com/" target="_self">Masuk ke youtube.</a>


_blank

Hasil :

Masuk ke youtube.

_self

Hasil :

Masuk ke Google.

Nah untuk Internal link kan kalian mengambil filenya di folder yang sama dengan file kalian. Tapi gimana kalau file kalian ada di folder berbeda.

Akan aku beri contoh jika file yang akan kalian tuju ada di file yang berbeda :

<a href="nama_folder_kalian/nama_file_kalian.html">Masuk ke latihan 2</a>


Hasilnya :

Masuk ke latihan 2

Untuk tanda "/" di kode yang aku kasih tadi itu namanya slash. Tanda slash ini berfungsi untuk mencari file didalam suatu folder. Tapi jika file yang akan kalian tuju ada diatas folder kalian gimana cara untuk menuju ke filenya. Caranya gampang, kalian hanya harus menambahkan "../". Mari kita lihat kodenya :



<a href="../1.html">Kembali ke latihan 1</a>
