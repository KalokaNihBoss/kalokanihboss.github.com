Pendahuluan

HTML adalah singkatan dari Hypertext Markup Language. HTML adalah bahasa Markup, bukan bahasa pemrograman. HTML ini pertama kali diciptakan oleh orang bernama
"Tim Berners-Lee". Orang ini juga adalah orang yang pertama kali menemukan :
- HTTP
- HTML
- WWW

Orang ini juga adalah orang yang pertama kali menciptakan :
- Web Browser pertama
- Web Server pertama
- Web Page pertama

Orang ini juga menjadi direktur dari sebuah organisasi atau konsorsium bernama W3C atau World Wide Web Consortium.



Jika kalian sudah membaca artikel tadi sekarang persiapkan bahan untuk membuat websitenya :

  • PC / Laptop * sedikit kuat
  • Kode Editor * terserah
  • Web Browser * terserah

Untuk kode editor aku pakai Visual Studio Code

Kalo web browser yang aku pakai adalah Chrome

Sebelum masuk materi ada satu hal yang akan aku bahas yaitu command. Command atau sering disebut komentar adalah suatu baris kode yang anya bisa dibaca di file secara langsung. Komentar ini tidak akan muncul di website. Lalu apa fungsi komentar?. Fungsinya untuk mengingat suatu baris kode. Jadi jika kamu punya website yang sudah lama tidak dilanjutkan kamu masih ingat baris dari suatu kode itu fungsinya untuk apa. Untuk memperjelasnya kita lihat kodenya saja :



<!-- Ini adalah komentar -->

Hasilnya adalah kosong karena seperti pada pengertiannya, komentar tidak akan muncul di website.
