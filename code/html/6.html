Table

Untuk membuat tabel kalian hanya perlu tag "Table" yang diberi atribut border, cellpadding, dand cellspacing

Didalam tag table kalian bisa memberi tag "td", "tr", dan "th". tag tr untuk membuat body table. Sementara tag th untuk membuat header table. Dan tag td untuk membuat isi table

Langsung aja kita lihat kodenya :



<table border="1" cellspacing="0" cellpadding="5"> <tr> <td>No.</td> <td>Nama</td> <td>Umur</td> </tr> <tr> <td>1.</td> <td>Kaloka</td> <td>11 Tahun</td> </tr> </table>


Hasil :

No. Nama Umur
1. Kaloka 11th
